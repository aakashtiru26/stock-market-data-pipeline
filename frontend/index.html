THE PIPELINE PROCESS

01

EXTRACTION LAYER

Connects to Yahoo Finance to fetch historical and real-time OHLCV data. Handles rate limits, timeouts, and batching to ensure data integrity before passing to the transformation engine.

02

TRANSFORMATION ENGINE

Utilizes Pandas to clean missing rows, align time-series indices, and compute derived technical indicators (SMA, EMA, MACD, RSI) entirely in memory.

03

RELATIONAL LOADING

Commits processed data securely to a local SQLite database using SQLAlchemy, providing a fast and reliable relational storage layer.

04

RESTFUL SERVING

FastAPI exposes the aggregated statistics, historical pricing, and calculated technical indicators via blazing fast, documented endpoints.
